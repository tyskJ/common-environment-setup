Git Bash インストール手順
=====================================================================

Windows
===================
1. インストーラダウンロード
--------------------------
`git for windows <https://gitforwindows.org/>`_ からインストーラをダウンロード

2. インストーラ実行
--------------------------
インストーラを実行します。※基本デフォルトで問題ありません。
各設定が気になる方は参考資料に載せているブログ記事を参照してください。
